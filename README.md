## 1

•  Створіть файл HTML з елементом <script>.
•  Використовуйте fetch для завантаження JSON-файлу з URL зображення.
•  Створіть елемент <img> з URL зображення.
•  Додайте елемент <img> на веб-сторінку.

## 2

Напишіть програму, яка перевіряє, чи даний текст є дійсним JSON.
• Використовуйте вбудовані функції мови програмування або бібліотеки для перевірки JSON.
• Виведіть результат перевірки (дійсний JSON / недійсний JSON).

## 3

Розробіть веб-додаток інтерактивну дошку завдань, на якій користувачі можуть додавати завдання та присвоювати їм теги або мітки.

## 4 (one of)

1. Створіть новий компонент в який інтегруйте роботу з публічним API (наприклад, JSONPlaceholder) та виведіть реальні дані на вашому веб-сайті. Інформація має бути дотичною до вашого завдання з рівня 3.
2. Додайте можливість фільтрації, сортування і пошуку.
3. Додайте різні типи візуалізації (графіки, діаграми, таблиці) ваших даних
4. Зробіть ваш веб-сайт доступним для людей з інвалідністю, використовуючи принципи ARIA та WCAG.
5. Доповніть ваш додаток можливістю реєстрації та управління профілем користувача (зміна пароля, інформації).
6. Для кожного елемента у вашому списку додайте можливість переходу на окрему сторінку, де будуть відображені деталі цього елемента.
7. Доповніть ваш веб-додаток пагінацією, щоб розділити дані на сторінки. Надайте можливість вибору кількості елементів, що відображаються на 1 сторінці - користувачу.
8. Розширте свій додаток, додавши форму, де користувачі можуть ввести дані для нового елемента(в новому компоненті)
9. Додайте можливість взаємодії з популярними сторонніми сервісами через API, наприклад, сервісами геолокації, соціальних мереж або платіжних систем.
10. Розширте ваш додаток, додавши підтримку різних мов та локалей.
